# Baseline allowlist for scripts/lint-app-infra-imports.sh — issue 049
# / ADR-029.
#
# Each entry is a file under internal/app/ or internal/cli/ that
# currently imports raioz/internal/{docker,proxy,orchestrate}
# directly instead of going through the segregated domain ports
# (interfaces.ContainerManager, ProxyManager, etc.).
#
# Rules:
#   - NEW files importing those packages directly fail the lint.
#   - REMOVING a file from this list happens when its imports route
#     through the port instead. Update in the same PR.
#   - Test files are NOT tracked (the lint skips _test.go).
#
# To regenerate the live list:
#
#   grep -rln '"raioz/internal/docker"\|"raioz/internal/proxy"\|"raioz/internal/orchestrate"' \
#       internal/app internal/cli | grep -v _test.go | sort
